Данная программа представляет собой реализацию розыгрыша игрушек в магазине детских товаров с использованием Java и принципов ООП.

Основной функционал программы включает:

    Создание класса Toy, который представляет игрушку и имеет следующие свойства:
        id: уникальный идентификатор игрушки;
        name: название игрушки;
        quantity: количество игрушек в магазине;
        weight: шанс выпадения игрушки (вес в % от 100);
        initialQuantity: начальное количество игрушек.

    Создание класса ToyStore, который представляет магазин игрушек и содержит список доступных игрушек. Он имеет следующие методы:
        addToy(Toy toy): добавляет игрушку в список магазина;
        getToyById(int toyId): возвращает игрушку по ее идентификатору;
        getToys(): возвращает список доступных игрушек;
        getTotalQuantity(): возвращает общее количество игрушек в магазине;
        hasEnoughToys(int numToys): проверяет, достаточно ли игрушек для проведения полноценного розыгрыша.

    В основном классе Main происходит основная логика программы. Он содержит методы для следующих действий:

        startRaffle(ToyStore toyStore, Scanner scanner): начинает розыгрыш игрушек. Пользователю предлагается ввести количество игрушек для розыгрыша. Если количество некорректно или не хватает игрушек для полноценного розыгрыша, программа выводит соответствующее сообщение. Затем случайным образом выбираются игрушки из списка доступных, и для каждой игрушки генерируется случайный идентификатор победителя. Результаты розыгрыша сохраняются в файл winners.txt.

        getRandomToy(List<Toy> toys, Random random): возвращает случайную игрушку из списка доступных игрушек на основе их шанса выпадения.

        generateWinnerId(Random random): генерирует случайный идентификатор победителя.

        updateToys(ToyStore toyStore, Scanner scanner): позволяет пользователю изменять количество и шанс выпадения игрушек. Выводится список доступных игрушек с их текущим количеством и шансом. Пользователь выбирает игрушку по идентификатору и вводит новое количество и шанс выпадения.

        resetToyQuantity(ToyStore toyStore): сбрасывает количество игрушек до начального значения.

    В методе main основного класса Main происходит основной цикл программы. Пользователю выводится меню с доступными действиями. Он выбирает номер действия, и программа выполняет соответствующую функцию или выходит из программы при выборе "0".

Таким образом, программа позволяет пользователю проводить розыгрыш игрушек в магазине, изменять количество и шанс выпадения игрушек, а также просматривать их текущую информацию.
